# @bitrix24/b24icons-nuxt

`@bitrix24/b24icons-nuxt` library for Vue 3.

## Installation

Install the module to your Nuxt application with one command:

```bash
npx nuxi module add @bitrix24/b24icons-nuxt
```

That's it! You can now use `@bitrix24/b24icons-nuxt` in your Nuxt app

Find more details in the [documentation](https://bitrix24.github.io/b24icons/docs/getting-started/installation/nuxt/).
